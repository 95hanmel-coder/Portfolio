# Hanita Melin – Portfolio

Modern developer portfolio built with HTML, CSS and JavaScript.

The portfolio showcases projects, technologies and cloud-focused development work with a clean responsive UI, dark mode support and smooth animations.

---

## Live Demo

Deployed with Vercel.

---

## Features

- Responsive design
- Dark mode with Local Storage persistence
- Smooth scroll navigation
- Animated scroll reveal effects
- Interactive project cards
- Mobile navigation menu
- Glassmorphism-inspired UI
- Modern gradient styling
- Fully static frontend

---

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Devicon
- Vercel

## Local Development

Clone the repository:

```bash
git clone https://github.com/95hanmel-coder/Portfolio.git
```

Navigate into the project:

```bash
cd Portfolio
```

Open `index.html` in your browser.
